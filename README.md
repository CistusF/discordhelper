Discord Helper
=============

디스코드 재설치 도우미 프로그램 입니다.
-------


사용법
======

메모장을 만드신뒤 해당 코드들을 복사, 붙여넣기 하신후
----------
파잃명을 설정 하신뒤 실행하시면 됩니다.
-------



다운 금지
==========

인코딩 오류가 생길수 있으므로 다운을 추천하지 않습니다.
-------

--------
